# 复习搭子

把带答案的复习资料自动整理成按章节练习的小测验。支持先查看带答案原资料，再进入选择题、判断题和多选题练习；支持自动批改、重置复习、只看正确答案和安装到设备，界面适配手机、平板与电脑。

## 运行

Windows 用户可以直接双击桌面的“复习搭子”快捷方式，它会在后台启动本地服务并打开浏览器。

不要直接双击 `index.html`。这是 Vite 项目的源码入口，必须通过本地服务访问。

也可以使用命令行：

```bash
pnpm install
pnpm dev
```

生产构建：

```bash
pnpm build
pnpm preview
```

## 安装到手机、平板和电脑

部署到 HTTPS 网站后，页面右上角会显示“安装应用”：

- Android 手机/平板：使用 Chrome 打开，点击“安装应用”
- iPhone/iPad：使用 Safari 打开，点击“分享”→“添加到主屏幕”
- Windows/macOS：使用 Chrome 或 Edge 打开，点击“安装应用”

安装后可以从桌面全屏启动；首次在线打开并完成缓存后，也可以离线使用。局域网中的普通 HTTP 地址只能用于浏览测试，手机安装需要 HTTPS。

### iPad 扫码离线安装

双击桌面的“复习搭子-iPad离线版”快捷方式，电脑会生成临时 HTTPS 地址和二维码：

1. 用 iPad 相机扫码，并选择使用 Safari 打开
2. 等待页面右上角显示“离线准备完成”
3. 点击 Safari 的“分享”→“添加到主屏幕”→“添加”
4. 从 iPad 桌面图标打开一次

完成后可以关闭电脑、断开网络继续使用。二维码只负责第一次下载安装应用文件；题库解析、答题记录和批改均保存在 iPad 本机。iPad 若清理了网站数据，需要重新扫码安装。

## 安卓 APK

同步网页代码并生成可直接安装的测试版 APK：

```bash
pnpm run android:apk
```

生成位置：`android/app/build/outputs/apk/debug/app-debug.apk`。这是供个人设备安装的调试签名版本；首次安装时，安卓可能会要求允许浏览器或文件管理器“安装未知应用”。

## 支持的资料

- 文件格式：TXT、Markdown、DOCX、PDF
- 章节标题：`第一章`、`第二单元`、`专题 1`、`Chapter 1`、Markdown 标题等
- 题目序号：`1.`、`1、`、`1)`、`（1）`
- 选项格式：`A. 选项`、`A、选项`，选项可逐行或写在同一行
- 答案格式：`答案：A`、`正确答案：AB`、`（√）`、`（错误）` 等

扫描版 PDF 目前不会进行 OCR；需要先把扫描图片转换成可复制文字，或粘贴识别后的文本。

## 数据与隐私

资料解析、答题和批改都在浏览器本地完成，不会上传到服务器。识别后的题库与答题进度保存在当前浏览器的 `localStorage` 中。
